# Goose Manager

基于 Python + PyWebView 的现代化文件管理器，拥有简洁美观的界面和实用的文件操作功能。

## ✨ 主要特性

- **多标签页** 管理多个目录，操作互不干扰  
- **双视图切换** 列表视图 / 图标视图，支持 Ctrl+滚轮快速切换  
- **文件操作** 复制、剪切、粘贴、删除、重命名、新建文件/文件夹  
- **拖拽支持** 拖拽文件到文件夹或驱动器进行移动/复制  
- **鼠标框选** + 键盘快捷键（Ctrl+A、Ctrl+C、Ctrl+V、Delete、F2 等）  
- **个性化设置** 深浅色主题、壁纸（默认渐变 / 必应每日 / 自定义图片）  
- **动画效果** 可调节速度与缓动，支持独立开关  
- **系统托盘** 关闭窗口时自动隐藏到托盘，常驻后台  

## 🛠 技术栈

- Python 3.8+  
- pywebview – 前端界面  
- pystray – 系统托盘  
- Pillow – 图标处理
